# mm
The comparative OBB 360 degree method.  
The guide of this should follow the instruction of MMRotate official code.  
The train.py in ./tools and config files are in the corresponding directory (./configs/model_name/my.py), to training the model (notice: the path need change):
```bash
python train.py
```
